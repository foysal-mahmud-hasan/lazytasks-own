== Installation ==

### How to Install LazyTasks

- **Download the Plugin**: Navigate to the 'Plugins' section in your WordPress dashboard, click 'Add New', and search for LazyTasks in the WordPress Plugin Directory.

-  **Install the Plugin**: Once you find LazyTasks, click the 'Install Now' button next to the plugin name. WordPress will automatically download and install the plugin for you.

-  **Activate LazyTasks**: After installation is complete, you'll need to activate LazyTasks to start using it. Find the plugin in your installed plugins list and click the 'Activate' link.

-  **Automatic Onboarding**: Once activated, go to the newly available LazyTasks section in your WordPress dashboard, you will be taken through our onboarding process to finish the installation process.

-  **Happy Tasking!**
